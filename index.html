¿Qué es un servidor?

Un servidor es una computadora que almacena páginas web y otros archivos. Cuando una persona entra a un sitio web, el servidor envía la información para que la página se muestre en el navegador.

¿Qué es el hosting?

El hosting es un servicio que permite guardar una página web en un servidor para que esté disponible en internet las 24 horas. Existen diferentes tipos de hosting como compartido, VPS, dedicado y en la nube.

¿Qué es un dominio?

El dominio es el nombre que identifica una página web en internet. Gracias al dominio las personas pueden entrar a un sitio fácilmente. Por ejemplo: google.com o youtube.com.

Relación entre dominio, hosting y servidor

El dominio es el nombre de la página web. El hosting es el servicio donde se guarda la página. El servidor es la computadora donde se almacenan los archivos del sitio web.
